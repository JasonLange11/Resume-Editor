# Resume Editor

A modern resume editor with live preview functionality built with React.

## Features

- **Live Preview**: See your resume update in real-time as you type
- **Personal Information**: Add your name, email, and phone number
- **Professional Summary**: Write a compelling professional summary
- **Work Experience**: Add multiple work experience entries with job title, company, duration, and description
- **Clean UI**: Modern, responsive design with a split-pane layout

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Usage

1. Fill in your personal information on the left side
2. Write your professional summary
3. Add work experience entries using the "Add Experience" button
4. Watch your resume preview update in real-time on the right side
5. Remove work experience entries if needed using the "Remove" button

## Technologies Used

- React 18
- Vite
- CSS3
